# Intel Image Classification

A deep learning image classification project built using TensorFlow and NumPy. The model is trained on the Intel Image Classification dataset to recognize different types of natural scenes.

## Project Overview

This project classifies images into multiple scene categories such as:

- Buildings
- Forest
- Glacier
- Mountain
- Sea
- Street

The goal is to automatically identify the category of a given image using a Convolutional Neural Network (CNN).

## Technologies Used

- Python
- TensorFlow
- NumPy
- Matplotlib

## Dataset

Intel Image Classification Dataset

The dataset contains thousands of labeled images belonging to different natural scene categories.

## Features

- Image preprocessing
- CNN-based image classification
- Model training and evaluation
- Prediction on unseen images
- Visualization of results

## Results

The trained model can classify natural scene images into their respective categories with good accuracy.
